# Quick Start

## 直接打开

在 Finder 或浏览器中打开：

- `kindergarten_storybook_mvp/frontend/index.html`

## 用本地静态服务打开

在仓库根目录执行：

```sh
python3 -m http.server 8081 --directory frontend
```

然后访问：

- `http://localhost:8081/`

## 当前版本包含什么

- 左侧老师录入表单
- 右侧 6 页故事卡预览
- 四个固定高频主题
- 浏览器打印 / 导出 PDF

## 当前版本不包含什么

- 用户登录
- 数据库存储
- 真实 AI 文本生成
- 真实 AI 插图生成
- 园所账号体系
