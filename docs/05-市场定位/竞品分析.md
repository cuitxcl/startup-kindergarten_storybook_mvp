# 竞品分析

## 目标

分析当前市面上与儿童定制读本相关的产品，明确：
- 是否已有类似产品
- 这些产品分别属于什么类型
- 哪些是直接竞品，哪些只是相近产品
- 当前产品真正的差异化机会在哪里

这份文档不追求覆盖所有个性化绘本产品，而是围绕当前项目最重要的判断：

`市场上已经有“孩子做主角的绘本产品”，但“面向幼儿园老师和园所、可批量生成个性化读本”的成熟标准化产品并不多。`

## 一、竞品分类

当前相关产品大致可以分为三类：

### 1. C 端个性化儿童绘本产品

核心卖点通常是：
- 孩子成为故事主角
- 家长购买个性化故事书
- 提供电子版或纸质版留存

这类产品已经证明“个性化绘本”本身存在需求。

### 2. 通用 AI 绘本创作平台

核心卖点通常是：
- 用户可以自由创作绘本
- 支持角色、情节、配音、插图等编辑
- 面向家庭、内容创作者或教育从业者

这类产品更像创作工具，不一定专门服务老师或幼儿园场景。

### 3. 学校或老师可用的故事生成工具

核心卖点通常是：
- 面向教师或学校
- 支持课堂故事、社会情绪故事、个别化内容
- 强调教学、支持性干预或班级使用

这类产品最接近当前项目方向。

## 三、竞品判断

## 已经被验证的需求

从现有产品可以看出，下面几件事已经被市场部分验证：

- 家长对“孩子做主角”有兴趣
- 个性化绘本可以收费
- 角色一致性是重要体验点
- 学校和老师场景也存在对故事生成工具的需求

## 尚未形成明显标准产品的位置

但当前更聚焦的这个位置，公开可见的成熟标准产品不多：

- 面向幼儿园老师
- 支持指定儿童替换主角
- 支持班级批量复用
- 支持围绕教学主题生成
- 支持园所和家园沟通场景

也就是说：

`“个性化绘本”不是空白市场，但“幼儿园老师/园所的批量个性化读本工具”仍有明显空位。`

## 四、当前产品与竞品的差异

当前产品如果只是做：
- 孩子做主角
- 自动生成故事
- 有插图

那会非常容易落入已有 C 端个性化绘本赛道，差异化不足。

当前产品真正应该强化的不是：
- “我们也能做孩子做主角的故事书”

而是：
- “老师能把它真正用进班级和家园沟通里”

## 五、可建立的差异化方向

建议重点强化下面几条差异化。

## 1. 面向老师真实工作流

不是做家庭娱乐产品，而是做老师真的会用的工具。

需要强调：
- 儿童档案
- 故事模板
- 教师可编辑
- 导出和分享
- 批量主角替换

## 2. 面向班级复用

相比单本定制绘本，更重要的是：
- 同主题不同主角快速复用
- 一个老师管理多个孩子
- 一个班级可以批量生成

这是 C 端个性化绘本产品通常不强的地方。

## 3. 面向家园沟通

相比只卖“绘本”，更应强调：
- 家长能收到和孩子相关的专属内容
- 园所能提升家长满意度
- 老师能更高效地做个性化反馈

## 4. 面向教育目标

相比纯故事娱乐，更应强调：
- 入园适应
- 分享合作
- 情绪表达
- 规则习惯
- 安全教育

这会使产品从“礼物型产品”变成“教育工具型产品”。

## 六、竞品威胁判断

## 直接威胁

真正的直接威胁不是所有个性化绘本产品，而是那些同时具备下面特征的产品：

- 支持孩子主角
- 支持老师或学校使用
- 支持稳定图片生成
- 支持批量生成和管理

目前公开可见的成熟产品不算多。

## 间接威胁

更大的间接威胁来自：
- 通用 AI 绘本创作平台
- 通用文生图工具
- 通用故事生成工具

因为用户可能会用这些通用工具拼出一个“够用方案”。

所以你的产品必须提供：
- 更低操作成本
- 更稳定角色一致性
- 更适合老师场景的流程

## 七、竞争机会

当前最值得抓住的机会不是：
- 再做一个 C 端个性化绘本产品

而是：
- 做一个面向老师和园所的个性化读本生产工具

这个机会点的核心在于：
- 使用场景更刚需
- 复用频率更高
- 商业模式更偏 B 端和家长增值，而不是单次消费

## 八、最终结论

现在市面上已经有类似产品，但大多属于：
- 家长购买的个性化绘本
- 通用 AI 绘本创作工具
- 部分学校或老师可用的故事生成产品

而你当前要做的更细分位置是：

- 面向幼儿园老师
- 面向班级使用
- 面向园所采购
- 面向家园沟通
- 支持孩子主角替换和批量复用

这个位置并不是完全空白，但公开可见的成熟标准化产品并不多，仍然有清晰的差异化机会。

一句话概括：

`“个性化绘本”赛道已有玩家，但“老师和园所可用的批量个性化读本工具”仍然存在机会。`
